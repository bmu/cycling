Dateien konvertieren

Wähle die HTML-Dateien eines Rennens aus. PDF- und andere Dateien werden automatisch ignoriert.

    So funktioniert es

    1. 1 HTML-Ergebnisdateien auswählen (z. B. je Rennen/Lauf eine Datei).
    2. 2 „Excel erzeugen" klicken — die Tabellen jeder Datei werden ausgelesen.
    3. 3 Die Datei ergebnisse.xlsx wird heruntergeladen — ein Arbeitsblatt je HTML-Datei.
